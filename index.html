Welcome to the Flashcard App

How does it work?

- Test yourself on a quiz category by clicking on the picture or scrolling down the page.

- Each time you click the "Randomise" button, a random question will be generated.

- Type your answer in the space provided and click "Submit answer" to see if you are correct.

- You can check the answer or remove the current question by clicking the appropriate button.

- You can add your own questions and answers by filling out the space provided and clicking "Add question".


Languages:

- Each language quiz has 10 pre-determined questions and answers which can be randomly generated.

- There are 10 bonus questions you can add to each quiz by clicking the "Add bonus word" button.


Mathematics:

- Each mathematics quiz contains 4 randomly generated questions, each using a different arithmetic operator (+, -, *, /).

- You can add as many questions as you like by clicking the "Add new" button containing the operator you wish to add.

- The difficulty increase for each quiz is determined by the maximum possible range of the digits in any given question.

(Beginner: max = 99, Intermediate: max = 999, Expert: max = 9999)


Create your own quiz:

- This quiz section has no pre-determined questions and is ready for you to add your own questions and answers.


Good luck!
